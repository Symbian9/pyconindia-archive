Building Real Time Django Apps

by Avinash Prasad (speaking)

Tweet
Section
Web Development
Session type
Talk
Technical level
Intermediate

Objective

The main objective is to enlighten the audience how Django can be used to build Real Time Applications. Integration of Djago with HTML 5 can bring about a whole new world of applications before us. The talk emphasizes how one can build Real Time Django Apps powered by HTML 5 and JavaScript.

Description

The session would explore the following pointers that would lead to development of Real Time Django Apps.

  1. What is a Real Time App?
  2. Powering Django with HTML5
  3. Django-SocketIO - What is it? & How it can be used to build real time app?
  4. A demo app using DjangoSocketIO, WebSockets.

Speaker bio

An avid Python programmer and tech geek. A Software Engineer by profession, have been working on Python/Django since 4 years. Also involved in mentoring/training professionals on Python as well.

I have been principally involved in designing/building web applications using Django and integrating them with various systems like LDAP, SMTP, IMAP servers. Also have been working on HTML 5, jQuery/JavaScript since sometime and integrating them with Django.
