Django templates for Web Designers

by Shreyank Gupta (speaking)

Tweet
Section
Web Development
Session type
Talk
Technical level
Intermediate

Objective

The objective is to provide UI and Web Designers with an overview of using the Django template language.

Description

Django is a high-level Python based web framework that encourages rapid development and clean, pragmatic design.

Django comes with it's very own templating layer, which is not merely python embedded into HTML, but it is a dumb python-like DSL which, by design, keeps programming logic out of the templates and keeps the presentation layer clean.

In this talk we will introduce ourselves to the Django Template Language and cover most of the useful tags and filters. We will also look into how to write custom tags and filters.

Speaker bio

Shreyank Gupta works as a Software Engineer at Red Hat. He programs in Python, Ruby and JavaScript.
