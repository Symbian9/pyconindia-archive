RPy2 : Using R from Python

by Harsh Singhal (speaking)

Tweet
Section
Scientific Computing
Session type
Talk
Technical level
Beginner

Objective

Introduce R as a Statistical Programming language and describe its use via Python by utilizing RPy2 ; A Python package which provides an interface to R

Description

  • Brief introduction to R as a Statistical Modeling environment
  • Using R from Python via RPy2
  • Interacting with R data structures in Python
  • Executing a few common Statistical functions via RPy2

Speaker bio

Harsh Singhal has used R both in academia and professionally for over 5 years.

He has used Python to engineer Predictive Modeling solutions by leveraging the Statistical tools available in R and using RPy2 to make them available in Python. He is currently a Data Scientist at LinkedIn, Bangalore.
