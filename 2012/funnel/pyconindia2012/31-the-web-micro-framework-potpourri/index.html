The web micro framework potpourri

by Arif Amirani (speaking)

Tweet
Section
Web Development
Session type
Talk
Technical level
Beginner

Objective

The intent is to expose attendees to the latest and greatest micro-framemworks available and provide enough insight on each one so as to be able to pick the right one for the job.

Description

This will be a detailed session with code demos and walkthroughs for all the major (and active) micro-frameworks available with a sample application. The attendees will thus be aware of:

  • The state of the projects
  • The pros and cons of each
  • Difficultly level of implementation
  • Any gotchas

I would like to limit the number of frameworks to 5 as I would like to cover some details within the allocated 45 mins. The current list of frameworks are:

  • Bottle
  • Flask
  • cherrypy
  • Bobo

If you would like to add more to the list above, please do comment.

Speaker bio

Currently an entrepreneur working on stealth projects primarily in Python and an overall fun guy! A few of my projects include:

  • APIKitchen.com - An API explorer that makes API development and testing very easy with support for snapshots for easy debugging.
  • Camper - Basecamp Android client
  • Stockmeter - Mac stock tracking application
  • Logcast - Cloud logging service (WIP)
