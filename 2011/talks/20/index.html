PyTI(PyPI Testing Infrastructure)

Talks | Submit a talk
Authors Yeswanth
Talk Type talk
Level Intermediate
Topic Testing
Tags pyti,pypi,gsoc
Summary

This talk describes about PyTI (Python Testing Infrastructure), a project started during GSoC(Google Summer of Code) 2011. The goal of the project is to test distributions from PyPI repository to assess quality and also to check if the distribution is malicious or not. Even though PyTI is not in production and still being developed , there are lot of design decisions which are firm and that is what this talk is going to deal with. Documentation of the project is provided here

Outline

Talk will discuss the following :

  • What is PyTI ?
  • How does PyTI benefit a python developer ?
  • Talk about the overall architecture PyTI uses and discuss the lifecycle of a package uploaded over PyPI and how it goes into testing
  • Move on to discuss the design - how Virtual Machines are handled, how and what tests are conducted,etc.
