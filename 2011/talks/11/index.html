Machine Learning Using Python

Talks | Submit a talk
Authors Vikram Kamath
Talk Type talk
Level Intermediate
Topic Scientific Programming
Tags Machine Learning, PyBrain, Scikits.learn
Summary

Machine Learning, although a relatively nascent field, has found itself to be an integral part of both academia and industry.The basic aim, is an introduction to Machine Learning,a few of the basic algorithms and the libraries in python for implementation of the algorithms.

Outline

The talk will emphasize on introducing some concepts of machine learning and some of the basic algorithms in Supervised and Unsupervised Learning along with Neural Networks.

I will be using PyBrain and Scikits.learn towards the end of the talk to demonstrate the abstractions that the libraries have to offer.

Notes

Please note that Support Vector Machines(SVM's) will not be included ,although some of the libraries offer the implementation of the same.

Profile of the authors

Vikram Kamath is a prefinal undergrad engineering student at Bangalore. He is working towards a Bachelors in Information Science and Engineering.He is currently an intern at the Department of Aerospace Engineering at the Indian Institute of Science and is working on using Learning Algorithms to classify sensor data.
