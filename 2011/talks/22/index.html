Django: REST in Peace

Talks | Submit a talk
Authors Shreyank Gupta
Talk Type talk
Level Intermediate
Topic Web programming
Tags Django, REST
Summary

The talk focuses on the need and importance of RESTful interfaces while developing Web Applications and how to achieve the same on Django.

Outline
  • A brief introduction to REST and RESTful interfaces.
  • Restful Views and URL's on Django, proper use of GET and POST.
  • MethodMiddleware to support PUT and DELETE.
  • A bit of un-obtrusive JavaScript.
Notes
Profile of the authors

Shreyank Gupta is a Web Developer by hobby and profession. Although a Ruby on Rails fanboy, Shreyank has over two years of experience coding on Python and Django. Shreyank blogs at http://allsortsofshrink.blogspot.com and tweets as @shreyankg
