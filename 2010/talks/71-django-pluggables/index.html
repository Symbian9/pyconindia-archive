Django Pluggables

Talks | Submit a talk
Authors Dheeraj Sayala
Level Intermediate
Topic Web programming
Tags django, web programming, reuse,
Summary

Python saves developer's time. Django's DRY (don't repeat yourself) philosophy is in line with it. Django pluggables can take it one step further.

This talk emphasizes the importance of building reusable django applications. It discusses the batteries included with django and some other pluggables out there. We take a look at the steps to create and share a pluggable.

Outline
  1. An introduction to reusable django applications(5 min)
  2. A tour of the existing pluggables. (10 min)
  3. Using pluggables (5 min)
  4. Creating reusable applications - an example (10 min)
  5. Distributing pluggables (5 min)
  6. Pointers to downloads and documentation (1 min)
  7. Questions
Notes

A knowledge of django and it's MTV architecture is required and assumed. People new to django can get started in a few minutes by attending a django introduction talk. So, it will be great if mine is scheduled after one such talk.
