Rapid 3D Game Development in Blender 2.5

Talks | Submit a talk
Authors Arun Ravindran
Level Beginner
Topic Game programming
Tags blender, 3d, game programming, game development, modeling, Python, AI
Summary

3D Game Development is a multi-disciplinary field today involving several man years of development. However, innovations in game design require tools to rapidly prototype and realise a game concept. Blender Game Engine is a free and open source game development tool which offers a end-to-end solution from modelling to scripting.

Learn how one can build a game framework within Blender without writing a single line of code. Extend the game in creative ways using Python scripting. The talk also covers several concepts of 3D game programming and where to look for further information.

Outline
  1. Introduction to 3D Game Programming
  2. Components of Typical 3D Games
  3. Python 3D Game Development
  4. Blender 2.5 Basics
  5. Blender Modelling Tutorial
  6. Blender Game Engine
  7. Game Tutorial in Blender
  8. Advanced Stuff
  9. Further Reading
Notes

The talk will cover some hands-on with Blender. Hence, it is recommended to install Blender 2.5 on your laptops before attending to fully appreciate the session.

Profile of the authors

Arun Ravindran has been an avid open source enthusiast and evangelist for over a decade. His language of choice is Python and occasionally dabbles in Haskell, Perl and Lisp. His favourite areas are Game Programming, Emergent Behaviour and Web programming. He regularly blogs at www.arunrocks.com

He is a functional consultant with a wide range of experience in Retail, Manufacturing and Banking working for a leading IT consulting company.
