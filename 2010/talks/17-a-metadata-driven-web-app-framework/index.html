A Metadata-driven Web App Framework

Talks | Submit a talk
Authors Rushabh Mehta
Level Intermediate
Topic Web programming
Tags web programming, framework
Summary

  1. The talk will introduce the "wnframework". The framework is a metadata driven, includes a rich front-end and has object-relational mapping.

  2. We will run through the motivations behind the framework and what makes it unique - high amount of automation and ability to build a high variety of apps

  3. The talk will contain a short demo to showcase the different features of the framework - metadata modeling, scripting, ORM etc

Outline
  1. Framework overview
  2. Quick comparison with other Frameworks
  3. Demo - Build an app
  4. Building the schema, scripting, object-relational mapping
  5. Questions and Feedback
Notes

  1. The framework is used in over 30 companies and we have recently launched an ERP SaaS product based on the framework (www.iwebnotes.com)

  2. The framework enables building of simple CMS based sites to complex web-apps like an ERP.

  3. It is developed from scratch and since 2004 and was first developed on top of Zope. The current version (4th) is based on cgi and runs in Apache. Database support is MySQL

  4. Maybe it is the first open source, rich-featured web app framework in India (though we don't claim it)

  5. Apart from Web Notes, two other software development companies are developing applications based on it - Sanchar Infotech (Mumbai) and New Indictrans Technologies (Pune).

  6. Is available under MIT license. and is hosted at http://code.google.com/p/wnframework

  7. Framework site: http://wnframework.org
