Automation using Python

Talks | Submit a talk
Authors Nitin Kumar, Swathi Nath
Level Intermediate
Topic GUI programming
Tags Pywinauto,sikuli,SWIG
Summary

This talk will give an idea how python and different tool developed over python can be used to automate any client Application over our Systems.

Outline

This talk will be around different tools being developed over python framework like pywinauto and sikuli. and how we can use these (or with modification of these tools) to automate any of our applications.

Any of the client application developed over C++ can be controlled using python. SWIG works as the bridge/Wrapper between C++ and Python and hence it Controls each wx widgets of the application.
