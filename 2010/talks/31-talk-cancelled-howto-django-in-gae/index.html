Talks | Submit a talk
Authors Venkatraman.S
Level Beginner
Topic Web programming
Tags Django, GAE
Summary

  • We will create a small app in Django and will deploy it on GAE.

  • Beginner knowledge in Django would suffice.

  • No fancy terms or lingo

  • We will see pros and cons of using GAE - for eg, how easy and cheap it is to deploy an app and get it running. [going 'lean']

  • Showcase http://www.mapatree.org/ which was developed in Django and is deployed on GAE.

Outline

The Summary explains the talk. This is for beginners and I dont intend to cover advanced topics. This is NOT a django beginners tutorial, but a getting started tutorial in GAE using Django.
