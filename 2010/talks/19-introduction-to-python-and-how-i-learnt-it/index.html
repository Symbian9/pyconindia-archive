Introduction to Python and How I learnt it

Talks | Submit a talk
Authors Kannappan
Level Beginner
Topic None of the above
Tags intro, newbie, hello world
Summary

This talk is a basic introduction to Python and is targeted at newbies. This talk will explain the basic constructs of the python programming language and touch up on some of the functional aspects of the language.

Outline

This talk will cover the below topics

  1. Hello World
  2. Numerical Operations
  3. Conditional Operations
  4. Strings
  5. Lists
  6. Dictionaries
  7. Loops
  8. Functions
  9. Functional aspects (map/reduce/filter/lambda)
Notes
Profile of the authors

Kannappan is a technical lead for Search Quality at MapQuest (AOL). He loves the elegance and conciseness of Python and uses it wherever possible. He tweets occasionally: @kannappan
