Indic Language Processing with SILPA

Talks | Submit a talk
Authors Santhosh Thottingal
Level Beginner
Topic Open source Python projects
Tags Natural Language Processing, Indic, Algorithms
Summary

Silpa project is an Indic Lanuage Processing toolkit. It is written in python and licensed as Free Software. The project aims to build a toolkit and framework to provide a common platform for all opensource Indic language projects. The project works as a web application, provides JSON RPC based web services, and it also works as a python library. Currently it supports around 15 language processing applications for 10+ Indian languages. Some of the algorithms and libraries were already used for other projects. Some of them even packaged in GNU/Linux distributions. Project: http://smc.org.in/silpa

Outline

The talk will cover the following:

  1. Introduce the SILPA framework- a custom python web framework based on WSGI
  2. Introduce the Indic language computing projects available in silpa project in brief with demos.
  3. Explain how silpa can be used: Python API, JSON RPC webservice, Web application

Profile of the authors

Santhosh Thottingal is a FOSS developer and enthusiast. He is the developer of many Indic Language Computing projects. He also works on accessibility with Indic Language and won FOSS India award 2008 for developing a TTS system for Indic Languages. He is one of top contributor of Swathanthra Malayalam Computing project which is upstream for more than 10 packages in various GNU/Linux Distros. Santhosh has conducted many Indic language processing related talks and workshops in various parts of India including FOSS.IN. More information about the author: http://thottingal.in/blog
