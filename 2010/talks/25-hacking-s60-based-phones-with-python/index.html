Hacking S60 based phones with Python

Talks | Submit a talk
Authors Steven Fernandez
Level Beginner
Topic Mobile computing
Tags Mobile apps
Summary

This talk will introduce the audience to installing and developing applications using python on the Symbian S60 series smartphone platform.

Outline
  • What is the Symbian S60 platform ?
  • Introduction to Python for the S60
  • How to install PyS60 ?
  • What can one use PyS60 for ?
  • How to develop PyS60 based applications ?
  • Sample PyS60 applications
  • Resources
Notes

This is just an introduction to whet the appetite of the audience for Python S60 Programming. There won't be anything 'hands-on' nor a detailed description of the PyS60 specific modules or api, although sample code and usage would be shown. Those who have already installed PyS60 and played with it a bit will not learn anything new.

Profile of the authors

Steven Fernandez is a developer who started off as a system adminstrator. He embraced python in his sys admin days back when Python was at version 1.5. His main tools these days are Python (~80% of all code he has written) and C/C++ on linux/Unix. Apart from the professional requirements, Python has been his language of choice for most personal tools.

He is quite passionate about Open Source and Linux is his platform of choice. Relevant to this talk is the fact that he recently became the maintainer of the Ensymble tools. He is using this talk as an excuse to spend more time with this project and also get more people involved in it.
