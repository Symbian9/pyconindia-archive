AULT : Agent based User simulation and Load Testing

Talks | Submit a talk
Authors Om Narayan
Level Intermediate
Topic Business applications
Tags Web Programming, Load Testing, User Simulation, social networking, AI
Summary

AULT, an application which I designed is used to test both User Interaction and Load Simulation of Social Networking web application's. Here agents act as users and they do specific tasks along with updating other users. It is capable of simulating a complete social networking web application and how its going to perform at a given time with the number of users along with task profile. This talk will be maily focused on, Why we need an Agent based simulation. We will also talk about how to design and implement agents in Python and how to scale agents on large scale. Most importantly We will also see how we can put A.I in agents and why simulate web application with fixed load ratio and why not let agents decide?

Outline

I am going talk about to following things . .

  • What are agents
  • what is 'Agent based User simulation and Load Testing'
  • How to design and implement agents in Python
  • How to scale agents on large scale
  • How we can put AI in Agents
  • Future of AULT
