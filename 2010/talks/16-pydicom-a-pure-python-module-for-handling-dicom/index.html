pydicom - a pure python module for handling DICOM

Talks | Submit a talk
Authors Arun B
Level Intermediate
Topic Open source Python projects
Tags DICOM,medical images,file parsing
Summary

I will start with an overview of the DICOM standard in medical imaging including the DICOM file format. I will then cover some details of the pure python dicom.py module for reading and writing DICOM files along with some examples. I will end with some suggestions for future extensions and interesting project ideas using dicom.py

Topics: 

* Overview of DICOM standard
* DICOM file format
* Introduction to dicom.py module
* Examples
* Future project ideas
Outline

Here is a rough break-up of the proposed talk

  1. Overview of DICOM standard (2 slides)
  2. DICOM file format description (2 slides)
  3. Current state of open source DICOM toolkits/libraries (1 slide)
  4. dicom.py module details (5 slides)
  5. Usage examples (2 slides)
  6. Interesting project ideas (1 slide)
