Creating and Deploying python web applications on Google App Engine

Talks | Submit a talk
Authors Saurabh Tiwari
Level Beginner
Topic Business applications
Tags newbie, GoogleAppEngine, Python2.x, python sdk, web application
Summary

The talk focuses on developing simple web-application on Google App Engine's Architecture. The talk will also focus on clarifying Google's Data Store Functionality and scripts for adding data into it. Since Google App Engine works with the Python runtime environment that uses Python version 2.5.2. so an explanation of the Usage will be described in the Talk.

Outline
  • Explain and give a presentation on google's cloud computing model, how is it different from the other cloud computing technology and the advantage of using it.
  • Briefing the Python SDK usage for Google App Engine.
  • About the Google Data Store.
  • Deploying Pre-existing applications over the GAE.
  • Creating a simple python based application on GAE localhost.
  • Deploying the created application on the Google App Engine.
  • Conclusion.
Notes

This talk/presentation/demo/tutorial will be a better opportunity for the attendees to learn if they come with their laptops and start deploying with the presenter.

Profile of the authors

Saurabh is a final year student and pursuing his B. Tech from Institute of Engineering and Rural Technology, Allahabad. He has been using python since a year, and his interest in web development makes him work on Google's App Engine. He has a clear understanding of building web applications on GAE using python SDK. He has been participating in App Engine's mailing list and developments from quite a long time. In his spare time he contributes to Melange.
