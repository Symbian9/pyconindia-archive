Natural Language Tool Kit

by Sree Ganesh (speaking)

Tweet
Section
Scientific Computing
Technical level
Beginner

Objective

Natural Language Processing is very interesting research field particularly Indian perspective We have implemented NLTK modules towards Indian languages. I am going to present NLTK (Python) modules implementation towards English and Indian Languages.

Description

NLTK is a leading platform for building Python programs to work with human language data. It provides easy-to-use interfaces to over 50 corpora and lexical resources such as WordNet, along with a suite of text processing libraries for classification, tokenization, stemming, tagging, parsing, and semantic reasoning. (nltk.org)

I will present practical implementation of NLTK, few basic NLTK modules, and some advanced NLTK modules like parsing, Namedentity recolonization and so on.

Requirements

No basic requirements for this talk - anyone can easily understand.

Speaker bio

Sre Ganesh Thottempudi is a developer with nearly 6 years of experience in NLP. He has been interested in Python for the last couple of years.
