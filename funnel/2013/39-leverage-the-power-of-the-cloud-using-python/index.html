Leverage the Power of the Cloud using Python

by Venu Murthy MK (speaking)

Tweet
Section
Workshops
Technical level
Beginner

Objective

I just had a successful presentation of this in the CloudCompute event in Mumbai last week. And hence would like to conduct a Workshop in which the attendees would be able to get their hands dirty by using Python to leverage the power of the cloud.

Description

Firstly the session will take them through a brief history of what is Cloud Computing and where exactly Python will come in handy. It will show them how popular Python is and how its influencing has been extending to the Cloud computing. We will see that there are client libraries available in Python for all the major cloud service providers like Google, RackSpace, Amazon EC2 and etc.
We will take one client library and use it to provision, update and purge VMs or Nodes

Requirements

If someone can boot a laptop and launch a terminal, they are welcome!

Speaker bio

Venu Murthy M K
Senior Cloud / SaaS Apps consultant, Pythonist
ThoughtWorks Technologies (India) Pvt Ltd

Venu Murthy has been fortunate to be the early adopters of Cloud technologies and hence has gained immense experience and knowledge on the Cloud Technologies. He has had the privilege to work on migrating large user bases along with their invaluable data on legacy systems to the Cloud. His decade long experience in all the phases of the cloud--from requirement gathering, analysis, spiking the cloud app to its deployment into the organisation and integrating it with other existing systems makes the cloud, magical! He is responsible for the most crucial cloud apps in ThoughtWorks (which has a sophisticated and large user base). He single handedly manages and monitors the cloud apps by automating most of the processes using the language he is passionate about--Python!
