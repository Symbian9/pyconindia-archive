PyCon India 2013

August 30 - September 1, 2013

(Skip ahead to session proposals)

The Fifth edition of PyCon India is being held in Bangalore,India from 30th August 2013 to 1st September 2013.

The organisers of PyCon India 2013 are looking for talk and workshop proposals to fill the formal presentation and workshop tracks. We accept proposals on a very broad range of topics related to Python programming.

http://in.pycon.org/2013/

Important Dates

  • Call for proposals opens: June 3, 2013
  • Proposal submission deadline: July 22, 2013
  • Proposal acceptance: July 31, 2013
  • First presentation upload: August 15, 2013
  • Final presentation upload (with changes if any): August 20, 2013

Status: Closed

Sections

  1. Core Python — Language features, Python implementations, Python 3.X, Standard Library
  2. Embedded Python — Embedded Python, Device Interfacing, Robotics
  3. Infrastructure — Deployments, System Configuration, Monitoring
  4. Scientific Computing — Scientific/Numeric Libraries, Cryptography, Data Engineering, Mathematics, Statistics, Visualizations
  5. Software Development Tools — GUI programming, Testing tools
  6. Web Development — Web Frameworks, Concurrency, Databases, NoSQL, Cloud Computing, Restful Interfaces
  7. Workshops — 3-hour long hands-on sessions on August 30.
