Let's talk testing with Selenium

by Anisha Narang (speaking)

Tweet
Section
Software Development Tools
Technical level
Beginner

Objective

To enable the participants to develop an effective approach to automated testing. This will include a brief on selenium, building and maintaining the test scripts and test report generation.
Also, an insight into some python modules will be included.

Description

Outline:
Why is testing important?
Brief on Selenium with Python as a testing framework
Best practices for automated testing
An approach to automated testing with a demo
How to maintain your test scripts?
Generate detailed HTML reports for the test runs

Focus on:
Testing the search functionality for any application, common UI features such as text highlighting, selecting checkboxes, text presence etc.
Understanding the UI elements and building test script

Requirements

Basic knowledge of python

Speaker bio

Anisha is currently working with Red Hat, Pune. She has done automated testing using Selenium with Python for one of the Red Hat products. Also, worked with various python libraries for testing purposes.
