Internet of Things - raspberry Pi and Python

by Yash Gajera (speaking)

Tweet
Section
Embedded Python
Technical level
Intermediate

Objective

After Attending this one can understand the concept of Internet of Things.
Benefits and Future application of Internet of Things.Also people can learn about the raspberry pi and internet of things embedded system using raspberry pi and Zigbee Technology.

Description

Session start with the logic Behind the word of Internet Of Things.
Than Concept of Internet Of things.explaining Whats is Raspberry PI?
Python programming In Raspberry pi.Internet of things using raspberry pi.
Demo Applications-live demo of controlling objects and monitor sensor data directly from Internet in form of Web Application.Talking future of Internet of things.

Speaker bio

Yash Gajera is final Year Electronics and communication engineering Student at AD Patel institute of Technology.Internet of Things Is his final year Project Which is going to be patent.He wrote python library For Zigbee communication with raspberry Pi that he is planing to open source in near future.Also he is planing a start up company working on internet of things.
