Werkzeug,Jinja2 and their implementation in flask

by Narasimha SV (speaking)

Tweet
Section
Web Development
Technical level
Beginner

Objective

Understanding of WSGI utility libraries Werkzeug and Jinja2 and how they are useful.
How these libraries are used to built a web framework.

Description

Below are the contents which i want to present.
1. Introduction to Wrekzeug.
2. Sample code.
3. URL Routing in Wrekzeug.
i.Rule Format ii. Builtin Converters iii.Maps, Rules and Adapters iv. Rule Factories v. Rule Templates. 4. HTTP Utilities in Wrekzeug
i. Date Functions ii. Header Parsing iii. Header Utilities iv. Cookies v. Conditional Response Helpers 5. Templating in Wrekzeug using Jinja2
i. Introduction to jinja2 ii. Sample jinja2 template code . 6. Implementation of Wrekzeug and Jinja2 in Flask.

Requirements

laptop with Ubuntu and python 2.7

Speaker bio

A techie working on cloud computing.Enthusiastic in learning new things. I have worked on many platforms from cloud to mobile using different technologies. Almost from past 3 years working on python related work and updating myself with newer technologies to learn things in faster manner.

This will be my first public session.
